Generate a Star system

(currently the only service avaiable)

Here there will be information about the Star system creator that I develop on my (limited) free time. The end goal is to produce a web page that produces a complete (fictional) star's celestial system with added info about as much as possible (inhabitants, politics, factions, various social metrics) to be used as a source of inspiration for rpg players or authors.
But it is mainly a place for me to test things while I'm learning new stuff, so the codebase is fluid and nothing to neither to learn from nor expect any coherence in.
So it is mainly created for my own amusement and as a learning opportunity, so warning for strange behaviour, sometimes incomplete implementations and frequent changes.

If Life is set to require the namegenerator will run and try a number of times and hopefully get a system containing life

This lists the major objects in the system. In addition to these there are thousands of small objects in irregular orbits.
